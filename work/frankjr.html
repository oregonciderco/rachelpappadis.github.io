Frank Jr's Day Off

In the spring of 2015, I created a short video game for a group project. To play the game download it here. Unzip the file and click on the .jar file. Make sure that you keep the jar file and the img folder in the same folder or else the file will not run properly.

For this project I did quality assurance and assisted with graphics and game concepts.

This project was done in collaboration with John Krakowski, Christ Waring, Ryan Stratton, Xu Yuanfei.
